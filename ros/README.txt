Simulate vehicles in Stageros using AgentDrive

1. Install ROS in version indigo
2. Replace world folder in /opt/ros/indigo/share/stage_ros
3. run roscore
4. run stageros using this command: rosrun stage_ros stageros $(rospack find stage_ros)/world/arena3.world
5. run AgentDrive configured for ROS simulation
6. Profit!!!!!

AD 5 use actual version of AgentDrive from branch kubesa with configuration in david.groovy
